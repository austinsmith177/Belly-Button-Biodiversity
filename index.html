Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

The interactive charts shown in this page help us to visualize the bacterial data for each volunteer. The drop down menu displays each of the volunteer's ID and by selecting this ID the charts get updated accordingly. Specifically, the volunteers should then be able to identify the top ten bacterial species in their belly buttons. That way if Improbable Beef identifies a species as a candidate to manufacture synthetic beef, the volunteers will be able to determine whether that species is found in their navel or not.
